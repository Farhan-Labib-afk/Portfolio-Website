
# Portfolio Website

A modern, responsive personal portfolio built with React, Vite, and Tailwind CSS.

## Features

- Curated sections for hero, about, experience, projects, research, skills, resume, and contact
- Responsive layout with reusable UI primitives
- Motion-friendly components and clean typography

## Tech Stack

- React + Vite
- Tailwind CSS
- Radix UI primitives
- MUI icons

## Getting Started

Install dependencies:

```bash
npm i
```

Start the dev server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

## Project Structure

- `src/app/App.tsx`: main page composition
- `src/app/components`: section components and UI building blocks
- `src/styles`: global styles and Tailwind setup

## Customization

- Update content in `src/app/components`
- Adjust colors, spacing, and typography in `src/styles`
